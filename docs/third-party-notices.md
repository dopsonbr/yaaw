# Third-Party Notices

This document tracks third-party icon sources referenced by YAAW's app-owned icon manifests.

## File Icon Manifests

- Material Icon Theme: MIT License, used as the default VS Code-inspired file-browser icon taxonomy.
- Catppuccin Icons: MIT License, used as the optional Dracula-friendly file-browser icon taxonomy.

The first icon-system implementation does not vendor upstream SVG files. It stores app-owned render descriptors and manifest metadata for the supported file and folder identities. If upstream artwork is vendored later, keep the upstream license file with the asset snapshot and update this notice with the exact source version.

## Product Icon References

- Codicons: used only as a VS Code product-icon reference for concepts such as explorer, source control, terminal, and panel controls.

Codicons are not vendored in the first version. If Codicons are bundled later, add the required attribution and license text before shipping.
