# Agent Brand Icons

YAAW displays compact agent identity icons for Codex, Claude, OpenCode, and GitHub Copilot so the sidebar can avoid repeated text labels.

The bundled resources are app-local identity marks fetched from public brand or product pages where available. They are only used to identify the selected user-owned CLI, are not written into user project directories, and do not imply endorsement by the brand owners.

Codex does not currently expose a dedicated downloadable first-party icon on the OpenAI brand page, so the bundled Codex mark comes from the LobeHub icon set entry that links back to the official Codex site. GitHub's current Copilot brand guidance says the old standalone Copilot icon is deprecated in favor of product lockups, but the compact sidebar slot uses the GitHub-authored standalone mark so the label remains recognizable at icon size.

Brand references:

- OpenAI brand guidance: https://openai.com/brand/
- Codex icon source: https://lobehub.com/icons/codex
- Anthropic company site: https://www.anthropic.com/
- Claude Code product page favicon: https://claude.com/product/claude-code
- GitHub Copilot brand toolkit: https://brand.github.com/brand-identity/copilot
- GitHub-authored Copilot SVG source: https://commons.wikimedia.org/wiki/File:GitHub_Copilot_logo.svg
- OpenCode project: https://opencode.ai/
